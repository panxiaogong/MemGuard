01
安装依赖
MemGuard 通过标准 HTTP 接口暴露服务，任何 HTTP 客户端均可接入。
pip install httpx
02
写入记忆
将 Agent 产生的记忆通过 MemGuard 写入，自动完成注入检测、PII 脱敏和 IMAG 免疫审查。
import httpx

resp = httpx.post("http://localhost:8000/v1/memory/write", json={
    "content":      "用户偏好深色主题",
    "source_id":    "agent-001",
    "source_type":  "USER_INPUT",  # USER_INPUT / EXTERNAL_URL / SYSTEM_GENERATED / TOOL_OUTPUT
    "session_hash": "session-abc",
    "trust_score":  0.8,           # 0~1，低于 0.5 触发额外审查
})
print(resp.json())  # {"status": "safe"} 或 {"status": "quarantined"}
03
查询记忆
语义检索，已隔离的条目不会出现在结果中。
resp = httpx.post("http://localhost:8000/v1/memory/read", json={
    "query":        "用户界面偏好",
    "session_hash": "session-abc",
    "n_results":    5,
})
for entry in resp.json()["results"]:
    print(entry["content"], entry["trust_score"])
